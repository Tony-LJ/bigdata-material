## 常用运维命令
### 查看哪个目录占用过高
```.text
cd到要查看的目录下，然后使用 du -h -x --max-depth=1 去查看哪个目录占用过高
du -h -x --max-depth=1
```
### 定时调度

# Example of job definition:

# |  |  |  |  |

分  时 日 月 周 [用户名]    <要执行的命令>
